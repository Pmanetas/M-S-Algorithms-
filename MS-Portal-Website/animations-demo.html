How to Use

Quick Start

Add the CSS file to your HTML:

<link rel="stylesheet" href="ms-animations-styles.css">

Then apply classes to your elements:

<div class="fade-in">This will fade in</div> <button class="ms-btn-hover-primary">Hover me</button> <div class="ms-card-hover">Card with hover effect</div>

Use CSS variables for consistent theming:

background: var(--ms-bg-primary); color: var(--ms-accent-primary); transition: all var(--ms-transition-base);
