Design Principles
  • Dark Theme: Always use #0f0f0f background
  • Green Accents: Use #16a34a for primary actions and highlights
  • Uppercase Labels: Navigation, buttons, and labels should be uppercase with increased letter-spacing
  • JetBrains Mono: Use for data, code, and technical content
  • Subtle Borders: Use rgba() values to maintain dark aesthetic
  • Hover States: Always provide visual feedback (color/shadow/transform)
  • 4px/8px Grid: All spacing should be multiples of 4px
Framework Integration

React/Next.js:

import './ms-design-system.css';

export default function Button({ variant = 'primary' }) {
  return <button className={`btn btn-${variant}`}>Click</button>;
}

Vue.js:

<template>
  <button :class="`btn btn-${variant}`">
    <slot />
  </button>
</template>
